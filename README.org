#+title: Stud.IP-HS-Merseburg-Layout-Plugin

* Beschreibung

Das HsMerseburgLayout-Plugin passt die Oberfläche, das User Interface und die Funktionalität eines Stud.IP an die Vorgaben der HS Merseburg an, sodass es als Basis für den Audiotranskriptionsdienst geeignet ist.

* Systemanforderungen

- Stud.IP ab Version 6.0
- SpeechToTextPlugin
